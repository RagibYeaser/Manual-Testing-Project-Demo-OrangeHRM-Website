**Testing Phase:** Functional | UI | Compatibility

## 🎯 Objective

To ensure the OrangeHRM system delivers a reliable and user-friendly experience by validating core functionalities such as authentication, employee management, leave processing, and admin operations. The testing focuses on identifying defects, ensuring data accuracy, and maintaining UI consistency.

---

## 🔑 Key Features Validated

### 🔐 User Authentication (Login)
- Valid and invalid login
- Logout functionality
- Input validation

### 👤 Employee Management (PIM)
- Add employee
- Search employee
- Edit employee details
- Delete employee (if applicable)

### 🏖 Leave Management
- Apply for leave
- View leave status
- Validate leave dates and inputs

### ⚙️ Admin Panel
- User management
- Role assignment
- System configuration
